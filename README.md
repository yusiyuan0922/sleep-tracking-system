# 睡眠小程序 - 失眠患者用药跟踪系统

## 项目结构

- `backend/` - Nest.js后端服务
- `miniprogram/` - uni-app小程序(患者端+医生端)
- `web-admin/` - Vue 3 Web管理后台
- `shared/` - 前后端共享类型定义
- `docs/` - 项目文档

## 技术栈

### 后端
- Node.js + Nest.js
- PostgreSQL 14+
- Redis
- TypeORM

### 前端
- uni-app (小程序)
- Vue 3 + Vite (Web管理后台)
- Element Plus
